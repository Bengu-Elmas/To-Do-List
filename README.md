# ✹ To-Do List

A simple and colorful to-do list application created with HTML, CSS, and JavaScript.

This project helps users organize their daily tasks, keep track of what they need to complete, and enjoy the satisfaction of checking items off one by one. It was created as a practice project to improve my front-end development and JavaScript skills.

## Preview



## Features

- Add new tasks to the list
- Mark tasks as completed
- Remove tasks from the list
- Prevent empty tasks from being added
- Simple and user-friendly interface
- Responsive input and task-list layout
- Colorful gradient background
- Custom buttons, shadows, borders, and list symbols

## Technologies Used

- **HTML5** for the page structure
- **CSS3** for styling and responsive design
- **JavaScript** for task creation, completion, and deletion

## How It Works

1. Enter a task into the text box.
2. Click the **Add** button.
3. The new task will appear in the list.
4. Click a task to mark it as completed.
5. Click the **×** button to remove it from the list.
